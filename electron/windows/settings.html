获取 Gemini API Key 的步骤
官网入口：https://aistudio.google.com/apikey
  1. 点击“打开 Google AI Studio”，使用 Google 账号登录。
  2. 进入 API Keys 页面后点击“Create API key”。
  3. 复制生成的密钥，回到此处粘贴到输入框。
  4. 点击“测试连接”，确认 Key、代理和 Gemini Live 模型可用后保存。
如果你在中国大陆网络环境下使用，建议先开启系统代理；代理输入框留空时 LiveScreen 会优先尝试系统代理或直连。
